API Dökümanı

DHTU API, Cilt hastalıklarına dair sınıflandırma sonuçları için programatik erişim sağlar. Resim dosyaları (jpg, png) veya Base64 ile kodlanmış veriler ile sınıflandırma sonucu için HTTP isteğinde gönderebilirsini.

API'a erişim için API key (Bearer Token) alınması gerekmektedir.
API key edinmek için lütfen bizimle iletişime geçin enessbostann@gmail.com || caldefne25@gmail.com || irmakderici516@gmail.com

POST - LOGIN

API Token bilginizi almak için aşağıdaki url'ye bir POST isteğinde bulunmanız gerekir :
http://34.118.92.185/api/login


QUERY PARAMETERS

Parametre Tür Açıklama
E-Posta Adresi String Kayıt olduğunuz mail adresi
Parola Stirng Belirlediğiniz Parola

Örnek İstekler (HTTP Requests)

JS

const data = new FormData();
data.append("email", "mailadresiniz@...com");
data.append("password", "parolanız");
const xhr = new XMLHttpRequest();
xhr.withCredentials = true;
xhr.addEventListener("readystatechange", function () {
if (this.readyState === this.DONE) {
console.log(this.responseText);
}
});
xhr.open("POST", "http://34.118.92.185/api/login");
xhr.send(data);

Java

HttpResponse response = Unirest.post("http://34.118.92.185/api/login")
.header("Content-Type", "multipart/form-data; boundary=---011000010111000001101001")
.body("-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"email\"\r\n\r\nmailadresiniz@...com\r\n-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"password\"\r\n\r\nparolanız\r\n-----011000010111000001101001--\r\n")
.asString();

POST - PREDICT WITH IMAGE FILE

Resim dosyası ile cilt hastalıklarına ilişkin tahmin sonuçlarını almak için aşağıdaki url'ye bir POST isteği göndermeniz gerekir:
http://34.118.92.185/api/image


QUERY PARAMETERS

Parametre Tür Açıklama
bearer_token String API keyiniz
image File jpg veya png formatında resim dosyası

Örnek İstekler (HTTP Requests)

JS

const data = new FormData();
data.append("image", "/IMG_4895.png");
const xhr = new XMLHttpRequest();
xhr.withCredentials = true; xhr.addEventListener("readystatechange", function () {
if (this.readyState === this.DONE) {
console.log(this.responseText);
} }); xhr.open("POST", "http://34.118.92.185/api/predict-with-image");
xhr.setRequestHeader("Authorization", "Bearer ...");
xhr.send(data);

Java

HttpResponse response = Unirest.post("http://34.118.92.185/api/predict-with-image")
.header("Content-Type", "multipart/form-data; boundary=---011000010111000001101001")
.header("Authorization", "Bearer ...")
.body("-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"image\"; filename=\"IMG_4895.png\"\r\nContent-Type: image/png\r\n\r\n\r\n-----011000010111000001101001--\r\n")
.asString();

PHP

$curl = curl_init();
curl_setopt_array($curl, [
CURLOPT_URL => "http://34.118.92.185/api/predict-with-image",
CURLOPT_RETURNTRANSFER => true,
CURLOPT_ENCODING => "",
CURLOPT_MAXREDIRS => 10,
CURLOPT_TIMEOUT => 30,
CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
CURLOPT_CUSTOMREQUEST => "POST",
CURLOPT_POSTFIELDS => "-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"image\"; filename=\"IMG_4895.png\"\r\nContent-Type: image/png\r\n\r\n\r\n-----011000010111000001101001--\r\n",
CURLOPT_HTTPHEADER => [
"Authorization: Bearer ...",
"Content-Type: multipart/form-data; boundary=---011000010111000001101001"
],
]);
$response = curl_exec($curl);
$err = curl_error($curl);
curl_close($curl);
if ($err) {
echo "cURL Error #:" . $err;
} else {
echo $response;
}

POST - PREDICT WITH BASE64

Base64 olarak kodlanmış veri ile cilt hastalıklarına ilişkin tahmin sonuçlarını almak için aşağıdaki url'ye bir POST isteği göndermeniz gerekir:
http://34.118.92.185/api/base64


QUERY PARAMETERS

Parametre Tür Açıklama
bearer_token String API keyiniz
image String Base64 ile encode edilmiş jpg veya png formatında resim dosyasına ait string veri

Örnek İstekler (HTTP Requests)

JS

const data = new FormData();
data.append("image", "data:image/jpeg;base64,/9j/...)
const xhr = new XMLHttpRequest();
xhr.withCredentials = true;
xhr.addEventListener("readystatechange", function () {
if (this.readyState === this.DONE) {
console.log(this.responseText);
}
});
xhr.open("POST", "http://34.118.92.185/api/predict-base64");
xhr.setRequestHeader("Accept", "application/json");
xhr.setRequestHeader("Authorization", "Bearer ...");
xhr.send(data);

Java

HttpResponse response = Unirest.post("http://34.118.92.185/api/predict-base64")
.header("Accept", "application/json")
.header("Authorization", "Bearer ...")
.header("content-type", "multipart/form-data; boundary=---011000010111000001101001")
.body("-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"image\"\r\n\r\ndata:image/jpeg;base64,/9j...)
.asString();

PHP

$curl = curl_init();
curl_setopt_array($curl, [
CURLOPT_URL => "http://34.118.92.185/api/predict-base64",
CURLOPT_RETURNTRANSFER => true,
CURLOPT_ENCODING => "",
CURLOPT_MAXREDIRS => 10,
CURLOPT_TIMEOUT => 30,
CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
CURLOPT_CUSTOMREQUEST => "POST",
CURLOPT_POSTFIELDS => '-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"image\..."'
CURLOPT_HTTPHEADER => [
"Accept: application/json",
"Authorization: Bearer ...",
"content-type: multipart/form-data; boundary=---011000010111000001101001"
],
]);
$response = curl_exec($curl);
$err = curl_error($curl);
curl_close($curl);
if ($err) {
echo "cURL Error #:" . $err;
} else {
echo $response;
}

Python

conn = http.client.HTTPConnection("34.118.92.185")
payload = '-----011000010111000001101001\r\nContent-Disposition: form-data; name=\"image\"\r\n\r\ndata:image/jpeg;base64,/...'
headers = {
'Accept': "application/json",
'Authorization': "Bearer ...",
'content-type': "multipart/form-data; boundary=---011000010111000001101001"
}

res = conn.getresponse()

print(data.decode("utf-8"))

HATALAR - ERRORS

DHTU API'ı aşağıdaki hata kodlarını kullanır:

Hata Kodu Açıklama
301 Resim dosyası jpg veya png formatında değil veya dosya boyutu 7MB'ın üzerinde.
302 İstekte resim dosyası yok
401 Unauthenticated: Bilinmeyen veya geçersizsecret_key. Bu hata, bilinmeyen bir API anahtarı kullanırsanız veya API anahtarınızın süresi dolmuşsa görünür.
401 email daha önceden kayıt edilmiş
400 password uzunluğu en az 6 karakter olmalıdır
